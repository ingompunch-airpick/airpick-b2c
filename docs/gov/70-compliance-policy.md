# 70 — 허위·과대·리뷰 조작 금지

코드 정본: `src/constants/complianceCopy.ts`  
공개 사실: https://www.에어픽.kr/facts/

## 절대 금지

1. 시드·기본 별점, 가짜 후기 수, 가짜 AggregateRating  
2. 「무조건 최저가」「확정 ○○원」「검증 완료」식 과대  
3. 에어픽을 **단일 주차장·발렛 업체**로 소개  
4. 미입점을 입점처럼, 보험·위치를 항상 보장처럼 단정  
5. 타인 시설·허위 현장 사진을 우리처럼 사용  

## 허용

- 예상·참고 요금 (일정·조건에 따라 달라질 수 있음)  
- 실이용 후기만 노출·정렬  
- 입점 = 예약·위치·사진·보험 **안내(제공 업체 기준)**  

## 채널

네이버 블로그·지식iN·카카오·플레이스·B2B 스토어 설명에도 동일.  
위반 발견 → 즉시 삭제·정정 + `/facts/` 문구로 맞춤.
