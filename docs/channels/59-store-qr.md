# 59 — 앱스토어·원링크·QR vs 웹 SEO

B2C 고객용 스토어 앱 **없음** (웹/PWA).  
B2B 앱 QR·원링크만 해당 → [57-b2b-store.md](./57-b2b-store.md)

## 규칙

- SEO 본문 랜딩(`/faq/`, `/parking`, `/guides/`)에 B2B 설치 배너를 **본문처럼** 올리지 않음  
- 캠페인 QR은 전용 단기 랜딩 또는 B2B URL + (필요 시) UTM  
- 고객용 공유 링크 기본값은 `https://www.에어픽.kr` 또는 해당 비교/가이드 URL
