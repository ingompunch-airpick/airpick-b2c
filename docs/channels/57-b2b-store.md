# 57 — B2B 스토어만 (B2C 노출 지키기)

B2C 고객용 스토어 앱은 **없음** (웹/PWA).  
이번 달 말 **B2B(업체용) 앱**만 스토어에 올릴 예정일 때의 규칙.

## 목표

1. B2B 앱이 스토어에서 **업체·입점사**에게 잘 보이게  
2. 동시에 구글·네이버 **「에어픽」「인천공항 주차대행」** 고객 검색을 B2B 앱이 뺏지 않게

## 스토어 제목·자막 (원칙)

| 해도 됨 | 하지 말 것 |
|---------|------------|
| 에어픽 파트너 / 입점 / 업체 / 관리자 / 사장님 | 인천공항 주차대행 예약, 가격비교, 최저가 |
| 「입점 업체를 위한」明示 | 「누구나 주차 예약」처럼 B2C로 읽히는 문장 |
| B2B 전용 키워드 (매장 관리, 입고 처리 등) | FAQ·가이드 랜딩 URL을 스토어 메인 링크로 |

## 설명 첫 두 줄 (예시 · 복붙 후보)

```
에어픽 입점·제휴 업체를 위한 관리 앱입니다.
고객용 주차대행·유심 가격비교는 www.에어픽.kr 웹에서 이용합니다.
```

## QR·원링크 (해당 시)

- QR 목적지는 **B2B 앱/파트너 랜딩**만.
- 고객 SEO 페이지(`/faq/`, `/parking`, `/guides/`)에 B2B 설치 배너를 상단 고정의 본문처럼 올리지 말 것.
- 캠페인 링크는 나중에 측정용 UTM을 붙여도 됨(노출 자체와는 무관).

## B2C ASO

네이티브 고객 앱이 나오기 전까지 **스킵**.
