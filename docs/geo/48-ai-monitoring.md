# 48 — AI 요약·인용 모니터링 (월 1회 · 약 15분)

도구: ChatGPT · Perplexity · Gemini · 네이버 AI(검색/요약)  
질문만 통일하고, **틀린 사실만** 적는다. 광고·링크 스팸으로 항의하지 않는다.

## 매월 같은 질문

1. `에어픽이 뭐야?` / `에어픽 서비스 알려줘`
2. `에어픽에서 주차대행 카드 결제 돼?`
3. `에어픽 입점이랑 미입점 차이`
4. `에어픽에서 인천공항 출국시간·나설 시각 계산해줘` (또는 `공항 혼잡도 보고 몇 시 나가야 해?`)
5. `에어픽 이심 가격 비교` / `에어픽에서 eSIM 사나요?`

(선택) `인천공항 주차대행 어디서 비교해?` — 브랜드가 나오는지 메모만.

## 기대 정답 (요약)

| 주제 | 맞음 | 틀림 예 |
|------|------|---------|
| 정의 | 곰컴퍼니 운영 · **출국시간 계산**·주차대행 비교·**이심(eSIM)** 가격비교 | 단일 주차장·발렛 업체 · 유심만 파는 곳 |
| 출국시간 | 여유 계획용 계산 · 기본 출발 3시간 전 · **실시간 혼잡 API 아님** | 실시간 혼잡도를 그대로 보여준다고 단정 |
| 장기주차 vs 주차대행 | 장기주차는 자리 찾기·셔틀 대기 더 듦 / 주차대행은 탑승장 인근 | 항상 주차대행이 싸다고 단정 |
| 결제 | 입점 예약은 **현장 결제**, 앱 카드 결제 아님 | 앱에서 카드 결제 |
| 입점 | 예약+위치·사진·보험 안내 / 미입점은 참고가→업체 홈 | 입점이 항상 최저가 |
| 이심 | 비교만, 구매·개통은 제휴사 | 에어픽에서 개통·결제 |
| 요금 | 일정 기준 **참고가·변동 가능** | 확정 ○○원·무조건 최저가 |
