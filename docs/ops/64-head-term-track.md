# 64 — 헤드텀 `인천공항 주차대행` 추적만

## 왜

단일어 「인천공항 주차대행」은 경쟁·광고·공식 안내가 세다.  
정면 SEO 글·허브를 더 쌓으면 **노력 대비 노출이 안 나오고**, 브랜드·비교 포지션이 희석될 수 있다.

## 정책

- **신규 랜딩·블로그 제목을 헤드텀 단독으로 쓰지 않음**
- 쓰려면 반드시 수식: `비교` · `가격비교` · `입점` · `T1/T2` · `보험` · `취소`
- 기존 `/parking` 은 「비교」로 유지 (이미 배포됨)

## 월간 기록 (GSC / 네이버)

| 년-월 | 노출 | 클릭 | 평균순위 | 비고 |
|-------|------|------|----------|------|
| | | | | |

순위가 올라도 **본진 콘텐츠를 늘리지 말고**, 비교·가이드·브랜드 쪽 CTR을 본다.
