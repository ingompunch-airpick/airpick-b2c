# 인천공항 1터미널·2터미널 주차, 뭐 기준으로 고르면 되나요?

에어픽은 곰컴퍼니가 운영하는 인천공항 출국시간 계산·주차대행 비교·이심(eSIM) 가격비교 플랫폼입니다.
고를 때는 항공편이 **어느 터미널인지**, 운서·집결 동선이 편한지가 먼저이고, 그다음 요금·실내/야외·보험을 보면 됩니다.

## 체크리스트
1. 항공권에 적힌 터미널(1터미널 / 2터미널)
2. 차량을 맡기는 시간대(야간 할증 여부)
3. 실내 주차 필요 여부
4. 입점인 경우 거리·보험·후기
5. 표시가는 입력 일정 기준 **참고가**

터미널 약어(T1/T2)보다 **1터미널·2터미널**로 기억해 두면 혼란이 덜합니다.
운서역·접근 동선은 업체마다 다르니, 비교 화면과 가이드를 함께 보세요.
