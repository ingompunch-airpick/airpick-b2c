# 해외여행 이심(eSIM), 공항에서 사기 전에 비교하려면?

에어픽은 곰컴퍼니가 운영하는 인천공항 출국시간 계산·주차대행 비교·이심(eSIM) 가격비교 플랫폼입니다.
에어픽은 **제휴 참고가를 나라·용량·일수·eSIM/USIM으로 비교**만 하고, 구매·개통은 제휴사 사이트에서 진행합니다.

## 초보가 먼저 알 것
1. 휴대폰이 eSIM을 지원하는지
2. 여행 일수·필요 데이터(또는 무제한)
3. 표시 가격은 참고가 (쿠폰·환율·옵션에 따라 달라질 수 있음)
4. 결제·개통·문의는 제휴사 고객센터

공항에서 급하게 사기 전에, 앱에서 가격순으로 한 번 걸러 두면 선택이 단순해집니다.
