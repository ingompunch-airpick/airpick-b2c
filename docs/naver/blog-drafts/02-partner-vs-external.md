# 에어픽 입점이랑 미입점, 뭐가 다른가요?

에어픽은 곰컴퍼니가 운영하는 인천공항 출국시간 계산·주차대행 비교·이심(eSIM) 가격비교 플랫폼입니다.
입점과 미입점은 **가격만의 차이가 아니라**, 예약·추적(위치·사진·보험)을 에어픽에서 받는지의 차이입니다.

## 입점
- 에어픽 앱에서 비교 후 바로 예약
- 입고 후 주차 위치·사진·보험 안내를 예약 탭에서 확인
- 이용 고객의 **실후기**만 노출 (가짜 별점 없음)

## 미입점
- 요금은 참고용으로만 비교
- 예약은 해당 업체 홈페이지로 이동
- 에어픽에서 위치·보험·후기를 제공하지 않음

## 오해하지 마세요
입점이 항상 더 싼 것은 아닙니다. 요금은 일정·터미널·실내/야외에 따라 달라집니다.
입점의 핵심은 **비교 후 예약·추적 흐름**입니다.
