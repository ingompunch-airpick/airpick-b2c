# 인천공항 주차대행, 보험은 어떻게 확인하나요?

에어픽은 곰컴퍼니가 운영하는 인천공항 출국시간 계산·주차대행 비교·이심(eSIM) 가격비교 플랫폼입니다.
주차대행을 맡길 때 **요금만 보지 말고 보험 가입 여부·안내 방식**도 함께 확인하는 것이 좋습니다.

## 바로 확인할 것
1. 업체가 보험 가입 여부를 안내하는지
2. 보장 범위·한도가 어디에 적혀 있는지 (홈·예약 안내·에어픽 입점 상세)
3. 입점이면 에어픽 예약 탭에서 보험 안내를 볼 수 있는지
4. 미입점이면 해당 업체 홈에서 직접 확인해야 하는지
5. 「무조건 보장」「100% 안심」 같은 **단정 문구**는 피하고, 안내·확인 수준으로 이해하기

## 한 줄 요약
에어픽 입점 업체는 비교·예약 후 **보험 안내를 예약 흐름에서 함께** 볼 수 있습니다. 미입점은 참고 요금만 비교하고, 보험은 업체 쪽에서 확인합니다.
