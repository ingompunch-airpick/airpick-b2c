에어픽 소개 배지 · 링크 가이드

입점사 홈·카톡에 「에어픽에서 비교·예약」을 붙이면, 손님은 비교로 오고 검색·신뢰에도 도움이 됩니다.

에어픽은 곰컴퍼니가 운영하는 인천공항 주차대행·유심·eSIM 가격비교 플랫폼입니다. 아래 배지·문구는 확정 요금·최저가 단정 없이 쓰도록 맞춰 두었습니다.

입점 업체 목록 주차 비교 허브

왜 붙이나 (짧게)

  • 손님에게 비교·예약 경로가 분명해집니다.
  • 에어픽 공개 페이지·www로 연결되면 검색·발견에 유리합니다.
  • 링크는 www.에어픽.kr 또는 내 업체 공개 URL로만 — 광고 디렉터리·유료 링크 구매는 하지 않습니다.

링크·문구 규칙 (컴플라이언스)

  • 허용: 비교·예약, 입점, 참고가, 일정 넣어 확인
  • 금지: 무조건 최저가, 확정 ○○원, 가짜 별점·후기, 「검증 완료」 과대
  • 가능하면 rel="sponsored" / nofollow 로 막지 말고 일반 소개 링크로 둡니다 (제휴 광고가 아닐 때).
  • 스팸성 디렉터리·유료 링크 구매는 하지 않습니다.

에어픽 쪽에서 할 일

  1. 입점사에 이 페이지 URL을 전달합니다.
  2. 붙인 뒤 홈·카톡에 링크가 살아 있는지 확인합니다.
  3. 새 입점이 생기면 /partners/{id}/용 스니펫을 이 페이지에 추가합니다.
